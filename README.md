# Creeper &middot; ![GitHub license](https://img.shields.io/badge/license-MIT-blue.svg)

Creeper 是一款 Minecraft Client 管理与部署工具

- 解耦化的管理服务器、服务端核心、模组
- 可视化部署过程
- 支持黑暗模式

## 调试

下载源码
```
git clone git@github.com:ShroXd/Creeper.git
```
安装依赖
```
yarn install
```
运行开发模式
```
yarn run electron:serve
```
